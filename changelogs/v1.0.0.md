# v1.0.0

This is the first release of Computer for Homey. You can wake, monitor, and shut down your computers from Homey, and use flows to automate those actions. The app supports multiple computers, each with its own network and SSH settings.

- Added Wake-on-LAN startup from Homey.
- Added online monitoring for configured computers.
- Added SSH-based shutdown support.
- Added Flow actions to start and shut down a computer.
- Added support for multiple computers with separate settings.
- Added OS selection for Windows, Linux, and macOS.
- Added configurable polling, SSH port, and broadcast address settings.
